# Space-Invaders
## Project Architecture L2 Licence Informatique 

Il s'agit d'une version simplifiée du jeu populaire "Space Invaders".


L’utilisateur déplace le joueur vers la droite (touche ’p’) ou vers la gauche (touche ’i’). En pressant la touche ’o’, il peut tirer un missile.

Le joueur a 3 vies.
